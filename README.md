# Vue2.0仿饿了么webapp单页应用
## 跟慕课网课程学习
## 依赖工具：
- vue-cli
- express
- vue-resource
- vue-router
- better-scroll
- stylus
- webpack
## 说明
安装node

```bash
# install dependencies
$ npm install

# server with hot reload at localhost:8080
$ npm run dev

# build for production with minification
$ npm run build
```
